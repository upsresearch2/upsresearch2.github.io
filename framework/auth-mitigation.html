Mitigation Strategies for Authentication Risks

Authentication allows only authorized users to access your smart home device.
  1. Check your device require an authentication credential, such as password, pin, and fingerprint, to access the device and its app?
    If so, set up one.
  2. If the device has a default username or password, change it to something you like.
  3. The username and password on both the device hardware and app should be complex. Here are some suggestions on strong password .
  4. Set up two-factor authentication, if allowed by the device. For example, the provider can generate a code by text.
